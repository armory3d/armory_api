spawnObject (name:String, parent:Object, done:Object ‑> Void, spawnChildren:Bool = true):Void

Spawn a new object instance in the Scene.

Parameters:

name

The String name of the Object as defined in blender.

parent

The parent object this new object should be attached to. (Optional use null to just add to the Scene without a parent).

done

A completion handler function to run after the spawn is complete. Example might want to change properties of the object after spawning.

spawnChildren

Also spawn the children of the newly spawned object. (Optional default is true).
